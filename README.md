# LoRa-Based Live Telemetry System — STM32WL Custom PCB

Custom PCB for real-time rocket flight telemetry using STM32WL with integrated LoRa radio.

## What It Does
Acquires live flight data from GPS, barometer, and IMU sensors and transmits wirelessly over LoRa to a ground station during launch.

## Hardware
- STM32WL — ARM Cortex-M4 with integrated LoRa sub-GHz radio
- GPS, barometric altimeter, and IMU sensors


## Tools
Altium Designer, STM32CubeIDE, Embedded C

## Context
Developed as part of the UTM Rocketry Team electronics and telemetry subsystem.
